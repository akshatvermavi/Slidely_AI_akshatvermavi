##  create a new repository on the command line
  
    git init
    
    git add README.md
    
    git commit -m "first commit"
    
    git branch -M main
    
    git remote add origin https://github.com/akshatvermavi/Slidely_AI_akshatvermavi.git
    
    git push -u origin main
